1. A domain name is a unique identifier used to locate a website on the internet.
It is a string of characters that is used to represent an IP address,
which is a numerical address that identifies a specific computer or server on the internet.
Domain names are typically composed of two parts:
 the Top-Level Domain (TLD) and the Second-Level Domain (SLD).
For example, in the domain name "sufficientrealestate.com," ".com" is the TLD and "sufficientrealestate" is the SLD.
Domain names are registered with domain name registrars, and can be purchased for a fee and renewed annually.

2. In computing, a server is a computer system or program that provides services or resources to other computers or devices on a network.
It can also refer to the physical hardware on which the software runs.
Servers are designed to handle requests from client computers, process data, and provide access to files, applications, or other resources.
For example, a web server is a computer program that delivers web pages to clients over the internet,
while a file server is a computer system that stores and provides access to files for clients on a network.
Servers can be dedicated to a single function or application, or they can be designed to handle multiple tasks simultaneously.
They can range from small, single-purpose systems to large, complex networks of servers working together to provide high-level services.

4. A uniform resource locator (URL) is a string of characters that is used to identify the address of a resource on the internet.
It is often referred to as a web address or a URL address.
A URL consists of several components, including a protocol (such as "http://" or "https://"), a domain name (such as "sufficientrealestate.com"),
and a path to the specific resource on that domain (such as "/search?q=uniform+resource+locator").
The URL serves as a unique identifier for the resource that it points to, and it allows users to easily access and share that resource over the internet.
URLs are used to locate web pages, images, videos, and other types of content on the World Wide Web,
and they are an essential component of the internet's infrastructure.

5. A domain name and a server are two separate components that work together to make a website accessible on the internet.
A domain name is the address that people use to find a website on the internet.
It is a human-readable name that corresponds to a unique IP address, which is a numerical identifier assigned to a device connected to the internet.
Domain names are purchased and registered through a domain name registrar and typically have a TLD (Top-Level Domain) such as .com, .org, or .net.
A server, on the other hand, is a computer or a network of computers that stores the website's files and serves them to users who request them.
Servers are responsible for processing requests and delivering web pages, images, and other content to the user's browser.
When someone types a domain name into their web browser, the browser sends a request to a Domain Name System (DNS)
server to look up the IP address associated with that domain name.
The DNS server then sends the IP address back to the browser, which uses it to connect to the server hosting the website.
Once the connection is established, the server sends the requested files to the browser, which displays them to the user.

So, in summary, a domain name is the address used to locate a website on the internet, while a server is the computer or
network of computers that stores the website's files and serves them to users who request them.
